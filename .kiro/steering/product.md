# Product Overview

เว็บแอปพลิเคชันร้านสั่งซื้อตุ๊กตาพรีออเดอร์ (Doll Pre-Order Web Application)

## Purpose
- ระบบ e-commerce สำหรับขายตุ๊กตาพรีออเดอร์จากแบรนด์ชั้นนำ (Jellycat, Care Bear, Steiff, Gund)
- รองรับการสั่งซื้อ, ชำระเงินผ่าน PromptPay QR, และติดตามคำสั่งซื้อ
- มีระบบ Admin สำหรับจัดการสินค้าและคำสั่งซื้อ

## Target Users
- ลูกค้าทั่วไปที่ต้องการสั่งซื้อตุ๊กตา
- Admin สำหรับจัดการระบบหลังบ้าน

## Key Features
- ค้นหาและเรียกดูสินค้า
- ตะกร้าสินค้าและ Checkout
- ชำระเงินผ่าน PromptPay QR Code
- ติดตามสถานะคำสั่งซื้อ
- Admin Dashboard (จัดการสินค้า, ผู้ใช้, คำสั่งซื้อ)

## Language
- UI และ content เป็นภาษาไทยเป็นหลัก
- Code comments และ API responses ใช้ภาษาไทย
